# Backend Absensi API

Sistem backend untuk aplikasi absensi dengan autentikasi JWT dan Firebase Realtime Database.

## ✨ Fitur

- ✅ Register & Login User
- ✅ Autentikasi JWT
- ✅ Check-in / Check-out
- ✅ Riwayat Absensi
- ✅ Manajemen User (Admin)
- ✅ Statistik Absensi
- ✅ Location Tracking
- ✅ Role Based Access Control

## 🛠️ Teknologi

- Node.js
- Express.js
- Firebase Admin SDK
- JWT untuk autentikasi
- bcryptjs untuk enkripsi password
- CORS enabled

## 📋 Prasyarat

- Node.js 14+
- NPM atau Yarn
- Akun Firebase dengan Realtime Database
- Service Account Firebase
